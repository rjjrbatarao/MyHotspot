# MyHotspot
Hotspot Admin Panel using laravel

##Features
```
- anti vpn
- multi hotspot
- mikrotik like features, profiles, interface, firewall, hotspot, pppoe
- charging and eloading feature
- sales report
- assignable global rates
- assignable profiles
- server side satellite
- open sattellite api
- open source
```
